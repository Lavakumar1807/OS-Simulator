System Calls

At a high level, a system call is a mechanism by which a program requests a service from the operating system's kernel. These services are typically related to hardware management, process control, file handling, memory allocation, or communication between processes. System calls are executed in privileged mode (kernel mode), ensuring that the operating system can control and monitor how user programs access system resources. The user program itself runs in user mode, which has restricted access to resources. This separation between user mode and kernel mode helps maintain system stability and security.
